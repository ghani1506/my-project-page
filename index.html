Project Information Background

The project "Teaching New Tricks to Academic Staff" is a collaborative initiative aimed at providing essential insights and best practices for academic staff at the University of Bristol (UoB). The project focuses on addressing the challenges posed by the COVID-19 pandemic in the delivery of teaching for coding and data science courses.

Purpose

The main purpose of this project is to compile and develop a comprehensive set of strategies and best practices to support academic staff in effectively delivering coding and data science courses in COVID-19 settings. One of the innovative strategies proposed is Game-based Pair Programming (GPP), aimed at enhancing student engagement and collaborative learning experiences.

Objectives

There are 2 objectives set for this project:

Objective 1 : To propose new trick called Game-based Pair Programming (GPP), which can be implemented to teach programming and data science during pandemic.

Objective 2 : To propose GPP in a short training plan for academic staff to be incorporated in their teaching pedgagogy.
